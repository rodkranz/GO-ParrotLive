# GO-ParrotLive 

An initiative, using frames from [`parrot-live`](https://github.com/hugomd/parrot.live), to allow any computer with `curl` to display an _animated_ party parrot.


# Dependency
```bash
go get -u github.com/golang/dep
```


# Run!
```bash
go run main.go
```


# Try it!
```bash
curl localhost:8080
```
